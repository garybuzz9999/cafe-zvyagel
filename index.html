🤖 Схема роботи:
1. Отримуєш фото меню на пошту
2. Завантажуєш фото в Claude (цей чат)
3. Claude розпізнає і дає JSON
4. Вставляєш JSON сюди і натискаєш "Імпорт"
📖 Мануал супер-адміна
🏪 Заклади
  • Список всіх закладів з рейтингом і кількістю відгуків
  • Видалити — прибирає заклад повністю зі всіма відгуками
  • Редагувати — змінити назву, адресу, години, фото
🤖 Імпорт меню через Claude
  1. Власник надсилає фото меню на твою пошту
  2. Завантажуєш фото в цей чат (claude.ai)
  3. Пишеш: "Розпізнай меню і дай JSON"
  4. Claude дає JSON зі стравами і цінами
  5. Копіюєш JSON → Адмін → 📋 Меню JSON
  6. Обираєш заклад → вставляєш → "Імпортувати"
  7. Меню з'являється на сайті автоматично
💡 Формат JSON: масив об'єктів з полями: cat, name, price, desc
💬 Відгуки
  • Всі відгуки по всіх закладах
  • Видалити — прибирає відгук що порушує правила
  • Антимат спрацьовує автоматично при публікації
🔐 Доступ

  • Пароль змінюється в коді (ADMIN_PASS_CAFE)
  • Власники закладів — окрема адмінка по телефону
